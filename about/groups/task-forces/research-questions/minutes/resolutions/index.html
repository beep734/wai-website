2022

  • 12 October: Accessible Platform Architectures Working Group Teleconference
    • Collaboration Accessibility User requirements. - RQTF requests APA conduct a Call for Consensus (CfC) to publish Collaboration Tools Accessibility User Requirements https://w3c.github.io/ctaur/ as a First Public Working Draft (FPWD).
  • 17 August: Accessible Platform Architectures Working Group Teleconference
    • Natural Language Interface Accessibility User Requirements. - RQTF requests an updated working draft publication of NAUR based on the current editors draft at https://www.w3.org/TR/naur/
  • 3 August: Accessible Platform Architectures Working Group Teleconference
    • w3C Accessibility Maturity Model. - RQTF requests that APA conduct a Call for Consensus (CfC) to publish the Maturity Model draft at https://w3c.github.io/maturity-model/index.html as a First Public Working Draft (FPWD), provided only that our Maturity Model subgroup resolves which (if any) implementing spread sheet should be referenced by this FPWD publication during its regularly scheduled subgroup teleconference later today, Wednesday 3 August.
  • 20 July: Accessible Platform Architectures Working Group Teleconference
    • Meeting minutes - RQTF requests updated working draft publication of the NAUR (https://www.w3.org/TR/naur/)
  • 29 June: Accessible Platform Architectures Working Group Teleconference
    • Accessibility of remote meetings. - RQTF recommends APA Call for Consensus to publish Accessibility of Remote Meetings (https://w3c.github.io/remote-meetings/) as a W3C Working Group Note.
  • 1 June: Accessible Platform Architectures Working Group Teleconference
    • Synchronization Accessibility User Requirements. - RQTF requests APA call for consensus to publish SAUR as found at https://raw.githack.com/w3c/saur/CFC-2022June/index.html as a W3C Note
  • 4 May: Accessible Platform Architectures Working Group Teleconference
    • Accessibility of Remote Meetings. - RQTF requests publication of second wide review draft of Accessibility of Remote meetings with comments due June 17th
  • 30 March: Accessible Platform Architectures Working Group Teleconference
    • Synchronization Accessibility User Requirements. - RQTF will publish SAUR https://www.w3.org/TR/saur/ for a second public review with comments due midnight Boston time Fri 13th May
  • 2 February: Accessible Platform Architectures Working Group Teleconference
    • Natural Language Interface Accessibility User Requirements. - group denies this one

2021

  • 1 December: Accessible Platform Architectures Working Group Teleconference
    • Revision of CAPTCHA Note. - RQTF recommends going to FPWD based on the editors draft https://raw.githack.com/w3c/captcha-accessibility/main/
  • 17 November: Accessible Platform Architectures Working Group Teleconference
    • Revision of CAPTCHA Note. - RQTF would very much like to meet the Flash mitigation folks in either Dec 1, 8 or 15 or Jan 5th, 12, 19 or 26
  • 3 November: Accessible Platform Architectures Working Group Teleconference
    • W3C Statements: Implications for Task Force publications. - RQTF will resume editing Inaccessibility of CAPTCHA with the goal of advancing to W3C statement status
  • 29 September: Accessible Platform Architectures Working Group Teleconference
    • Natural Language Interface Accessibility User Requirements. - RQTF requests APA call for consensus to publish a first public working draft (FPWD) of Natural Language Interface Accessibility User Requirements (NAUR) found at https://raw.githack.com/w3c/apa/9ee0e5e675e653c4299df3508070f4b1c5528977/naur/index.html with comments due by 12 November
    • Accessibility of Remote Meetings. - RATF requests APA call for consensus to publish a first public working draft of Accessibility of Remote Meetings found at https://raw.githack.com/w3c/apa/9c9109f52551b42e27f77a61b30415602df39565/remote-meetings/index.html with comments due 19 November
  • 1 September: Accessible Platform Architectures Working Group Teleconference
    • Synchronization Accessibility User Requirements. - RQTF requests APA conduct a Call for Consensus on publication of a First Public Working Draft of the Synchronization Accessibility User Requirements (SAUR) https://raw.githack.com/w3c/apa/main/saur/ once our Editor's Note is updated by agreement to address recently published guidance from the U.S. FCC
  • 28 July: Accessible Platform Architectures Working Group Teleconference
    • XR Accessibility User Requirements: issues and publication request. - RQTF requests APA issues a CFC to publish XAUR http://raw.githack.com/w3c/apa/458e82b2ce737120d774dc3f2f1d26fec070415f/xaur/index.html as a W3C NOTE
    • Planning on forthcoming publications - RQTF to prioritise a FPWD of SAUR followed by NAUR
  • 24 March: Accessible Platform Architectures Working Group Teleconference
    • RAUR: updates and issues. - RQTF requests APA issue a Call for Consensus on two documents to be published as W3C Notes, the RAUR and the XAUR.

2020

  • 18 November: Accessible Platform Architectures Working Group Teleconference
    • RAUR and XAUR: the plan to achieve W3C Working Group Note status - RQTF recommends APA publish the RAUR at https://raw.githack.com/w3c/apa/2020-nov-raur-publication/raur/index.html as a W3C Note.
  • 26 August: Accessible Platform Architectures Working Group Teleconference
    • RTC Accessibility User Requirements (RAUR) open issues. - It's an ongoing conversation that will bring up new scenarios
  • 19 August: Accessible Platform Architectures Working Group Teleconference
    • Miscellaneous updates. - To try the "Accessibility of Remote Meetings" as a Note

2019

  • 23 October: Accessible Platform Architectures Working Group Teleconference
    • CAPTCHA erratum. - RQTF recommends the errata update to our CAPTCHA note as described in https://lists.w3.org/Archives/Public/public-rqtf/2019Oct/0018.html.
  • 31 July: Accessible Platform Architectures Working Group Teleconference
    • CAPTCHA Note - update. - RQTF finds work on the CAPTCHA revisions is now complete and recommends APA publish the latest Editor's Draft as the new W3C Note on CAPTCHA.

2017

  • 24 May: Accessible Platform Architectures Working Group Teleconference
    • CAPTCHA findings - reporting to APA Working Group - Forward Captcha findings to APA WG, with proviso additional updates may be forthcoming.
