Uma árvore de decisão do atributo alt

em Images Tutorial (em Inglês)

About this translation

Esta tradução voluntária pode não refletir de forma adequada as intenções da versão original em ingles.

Tradutor: Adiel Cristo.
WAI agradeçe os/as tradutores/as, e dá as boas-vindas a outras traduções.

Visão geral

Esta árvore de decisão descreve como usar o atributo alt do elemento <img> em várias situações. Para alguns tipos de imagens, há abordagens alternativas, como usar imagens de fundo CSS para imagens decorativas ou fontes web em vez de imagens de texto.

  • A imagem contém texto?
    • Sim:
      • … e o texto também está presente como texto real próximo. Use um atributo alt vazio. Consulte Imagens decorativas (em Inglês).
      • … e o texto é mostrado apenas para efeitos visuais. Use um atributo alt vazio. Consulte Imagens decorativas (em Inglês).
      • … e o texto tem uma função específica, por exemplo, é um ícone. Use o atributo alt para comunicar a função da imagem. Consulte Imagens funcionais (em Inglês).
      • … e o texto na imagem não está presente de outra forma. Use o atributo alt para incluir o texto da imagem. Consulte Imagens de texto (em Inglês).
    • Não:
      • Continue.
  • A imagem é usada em um link ou botão e seria difícil ou impossível entender o que o link ou o botão faz se a imagem não estivesse lá?
    • Sim:
      • Use o atributo alt para comunicar o destino do link ou a ação tomada. Consulte Imagens funcionais (em Inglês).
    • Não:
      • Continue.
  • A imagem contribui com significado para a página ou contexto atual?
    • Sim:
      • … e é um gráfico ou fotografia simples. Use uma breve descrição da imagem de uma forma que transmita esse significado no atributo alt. Consulte Imagens informativas (em Inglês).
      • … e é um gráfico ou informação complexa. Inclua as informações contidas na imagem em outro lugar da página. Consulte Imagens complexas (em Inglês).
      • … e mostra conteúdo redundante do texto real próximo. Use um atributo alt vazio. Consulte Imagens funcionais (em Inglês) (redundantes).
    • Não:
      • Continue.
  • A imagem é puramente decorativa ou não é destinada às pessoas usuárias?
    • Sim:
      • Use um atributo alt vazio. Consulte Imagens decorativas (em Inglês).
    • Não:
      • Continue.
  • O uso da imagem não está listado acima ou não está claro qual texto o atributo alt deveria fornecer?
    • Esta árvore de decisão não abrange todos os casos. Para obter informações detalhadas sobre o fornecimento de alternativas de texto, consulte os Tutoriais de imagens (em Inglês).
